Plugin Leaflet.js

Mode d'emploi d'utilisation du plugin leaflet.js

1. Télécharger Leaflet

Aller sur le site officiel de leaflet.

Télécharger Leaflet et extraire les fichiers.

2. Intégrer Leaflet dans la page HTML

Dans la section <head> ajouter la feuille de style :
<link rel="stylesheet" href="leaflet/leaflet.css">

Avant la fin du <body> ajouter le script JavaScript de Leaflet et la page script :
<script src="leaflet/leaflet.js"></script>
<script src="script.js"></script>

3. Ajouter un élément HTML pour la carte

Dans le fichier index.html ajouter un <div> avec un id="map" pour afficher la carte :
<div id="map" style="height: 400px;"></div>

4. Initialiser la carte avec JavaScript

Dans le fichier script.js ajouter la fonction pour initialiser la carte Leaflet :
var map = L.map('id').setView([coordonée x, coordonée y], zoom);

Et la fonction pour afficher les "tiles" de la carte avec openstreetmap :
L.tileLayer('https://tile.openstreetmap.org/{z}/{x}/{y}.png?{foo}', { foo: 'bar', attribution: '© OpenStreetMap contributors' }).addTo(map);

Ajouter un marqueur

Dans le fichier script.js ajouter la fonction pour mettre un marqueur :
L.marker([coordonnée x, coordonnée y]).addTo(id);

Dans mon cas la fonction sera :
L.marker([45.166672, 5.71667]).addTo(map);

Ajouter une icon au marqueur

Dans le fichier script.js ajouter la fonction pour mettre un marqueur :
L.marker([coordonnée x, coordonnée y]).addTo(id);

Dans mon cas la fonction sera :
L.marker([45.166672, 5.71667]).addTo(map);
